## 1. Linux 텍스트 편집기

### 1.1 편집기의 종류

| 분류 | 편집기 | 설명 |
|------|--------|------|
| 줄 편집기 | `ed`, `ex`, `sed` | 한 번에 한 줄씩 편집 |
| 화면 편집기 | `vi`, `emacs` | 전체 화면 편집 (모달 / 모드리스) |
| GUI 편집기 | `gedit`, `kate` | 그래픽 인터페이스 편집기 |

### 1.2 모달 vs. 모드리스 편집기

| 기능 | 모달 (vi/vim) | 모드리스 (nano, notepad) |
|------|--------------|-------------------------|
| 입력 모드 | 텍스트 입력 | 텍스트 입력 |
| 복사 | `yy` (명령 모드) | `Ctrl+C` |
| 붙여넣기 | `p` (명령 모드) | `Ctrl+V` |
| 저장 | `:wq` 또는 `ZZ` | `Ctrl+S` |
| 모드 전환 | `i`, `a`, `o`, `Esc` | 해당 없음 |

> **핵심 포인트:** `vim`은 `vi`의 업그레이드 버전("Vi IMproved")이다. 핵심 사용법은 동일하다.

## 2. Vi 동작 모드

### 2.1 세 가지 모드

```
                    i, I, a, A, o, O
    +-------------+  ──────────────>  +-------------+
    | 명령 모드   |                   |  입력 모드   |
    | (Normal)    |  <──────────────  | (Insert)     |
    +------+------+      Esc          +-------------+
           │
      : / ?│         Enter
           v           │
    +------+------+    │
    | 마지막 줄   |────┘
    | 모드 (Ex)   |
    +------+------+
           │
     :q :wq :q!
           v
    +-------------+
    |   Vi 종료   |
    +-------------+
```

- **명령 모드 (Normal):** 키 입력을 명령으로 해석 (커서 이동, 삭제, 복사)
- **입력 모드 (Insert):** 키 입력이 텍스트 내용으로 입력됨
- **마지막 줄 모드 (Ex):** 화면 하단에서 `:`, `/`, `?` 뒤에 명령어 입력

## 3. Vi 시작 및 종료

### 3.1 Vi 시작

```bash
vi filename              # 파일 열기 (존재하지 않으면 생성)
vi                       # 빈 버퍼 열기
vi +10 filename          # 파일의 10번째 줄에서 열기
vi +/pattern filename    # 패턴의 첫 번째 위치에서 열기
vi -R filename           # 읽기 전용 모드로 열기
vim filename             # vim 사용 (향상된 vi)
```

### 3.2 저장 및 종료

| 모드 | 명령어 | 동작 |
|------|--------|------|
| 마지막 줄 | `:q` | 종료 (변경 사항이 없을 때만) |
| 마지막 줄 | `:q!` | 저장하지 않고 종료 |
| 마지막 줄 | `:w` | 저장 |
| 마지막 줄 | `:w filename` | 다른 파일로 저장 |
| 마지막 줄 | `:wq` | 저장 후 종료 |
| 마지막 줄 | `:wq!` | 강제 저장 후 종료 |
| 명령 | `ZZ` | 저장 후 종료 (Shift+z, Shift+z) |

## 4. 커서 이동

### 4.1 기본 이동

| 키 | 동작 |
|----|------|
| `h` | 왼쪽으로 한 문자 이동 |
| `j` | 아래로 한 줄 이동 |
| `k` | 위로 한 줄 이동 |
| `l` | 오른쪽으로 한 문자 이동 |
| 방향키 | h/j/k/l과 동일 |

### 4.2 줄 이동

| 키 | 동작 |
|----|------|
| `0` 또는 `^` | 줄의 시작으로 이동 |
| `$` | 줄의 끝으로 이동 |
| `-` | 이전 줄의 시작으로 이동 |
| `+` 또는 `Enter` | 다음 줄의 시작으로 이동 |

### 4.3 단어 이동

| 키 | 동작 |
|----|------|
| `w` | 다음 단어의 시작으로 이동 |
| `W` | 다음 단어로 이동 (공백 구분) |
| `e` | 현재 단어의 끝으로 이동 |
| `E` | 단어의 끝으로 이동 (공백 구분) |
| `b` | 이전 단어의 시작으로 이동 |
| `B` | 이전 단어로 이동 (공백 구분) |

### 4.4 화면 이동

| 키 | 동작 |
|----|------|
| `Ctrl+f` | 한 페이지 앞으로 (Page Down) |
| `Ctrl+b` | 한 페이지 뒤로 (Page Up) |
| `Ctrl+d` | 반 페이지 앞으로 |
| `Ctrl+u` | 반 페이지 뒤로 |
| `Ctrl+e` | 한 줄 아래로 스크롤 |
| `Ctrl+y` | 한 줄 위로 스크롤 |

### 4.5 특정 위치로 이동

| 키 | 동작 |
|----|------|
| `G` | 파일의 마지막 줄로 이동 |
| `1G` 또는 `gg` | 파일의 첫 번째 줄로 이동 |
| `nG` | n번째 줄로 이동 (예: `20G`) |
| `:n` | n번째 줄로 이동 (마지막 줄 모드) |
| `:$` | 마지막 줄로 이동 |
| `H` | 화면 상단으로 이동 |
| `M` | 화면 중간으로 이동 |
| `L` | 화면 하단으로 이동 |

## 5. 편집 명령어

### 5.1 입력 모드 진입

| 키 | 동작 |
|----|------|
| `i` | 커서 앞에 삽입 |
| `I` | 줄의 시작에 삽입 |
| `a` | 커서 뒤에 추가 |
| `A` | 줄의 끝에 추가 |
| `o` | 아래에 새 줄 열기 |
| `O` | 위에 새 줄 열기 |

### 5.2 텍스트 삭제

| 키 | 동작 |
|----|------|
| `x` | 커서 아래 문자 삭제 |
| `X` | 커서 앞 문자 삭제 |
| `dw` | 커서에서 단어 끝까지 삭제 |
| `dd` | 전체 줄 삭제 |
| `d$` 또는 `D` | 커서에서 줄 끝까지 삭제 |
| `d0` | 커서에서 줄 시작까지 삭제 |
| `ndd` | n줄 삭제 (예: `5dd`) |
| `dG` | 현재 줄에서 파일 끝까지 삭제 |
| `d1G` | 현재 줄에서 파일 시작까지 삭제 |

### 5.3 실행 취소 및 다시 실행

| 키 | 동작 |
|----|------|
| `u` | 마지막 변경 취소 |
| `U` | 현재 줄의 모든 변경 취소 |
| `Ctrl+r` | 다시 실행 (vim에서만) |
| `.` | 마지막 명령 반복 |

### 5.4 텍스트 대체

| 키 | 동작 |
|----|------|
| `r` | 커서 아래 단일 문자 대체 |
| `R` | 대체 모드 진입 (덮어쓰기) |
| `cw` | 단어 변경 (단어 삭제 + 입력 모드 진입) |
| `cc` 또는 `S` | 전체 줄 변경 |
| `c$` 또는 `C` | 커서에서 줄 끝까지 변경 |
| `~` | 커서 아래 문자의 대소문자 전환 |
| `J` | 현재 줄과 다음 줄 합치기 |

## 6. 복사, 잘라내기, 붙여넣기

### 6.1 Yank(복사)와 Put(붙여넣기)

| 키 | 동작 |
|----|------|
| `yy` 또는 `Y` | 현재 줄 복사 (yank) |
| `nyy` | n줄 복사 (예: `5yy`) |
| `yw` | 단어 복사 |
| `y$` | 커서에서 줄 끝까지 복사 |
| `p` | 커서 뒤/현재 줄 아래에 붙여넣기 (put) |
| `P` | 커서 앞/현재 줄 위에 붙여넣기 (put) |

### 6.2 이름 있는 버퍼 (레지스터)

```
"ayy        # 현재 줄을 레지스터 'a'에 복사
"ap         # 레지스터 'a'에서 붙여넣기
"bdd        # 현재 줄을 레지스터 'b'에 삭제(잘라내기)
```

## 7. 검색 및 바꾸기

### 7.1 검색

| 명령어 | 동작 |
|--------|------|
| `/pattern` | 앞으로 패턴 검색 |
| `?pattern` | 뒤로 패턴 검색 |
| `n` | 같은 방향으로 검색 반복 |
| `N` | 반대 방향으로 검색 반복 |
| `*` | 커서 아래 단어를 앞으로 검색 |
| `#` | 커서 아래 단어를 뒤로 검색 |

### 7.2 검색 및 바꾸기 (치환)

```vim
:s/old/new/              # 현재 줄에서 첫 번째 항목 바꾸기
:s/old/new/g             # 현재 줄의 모든 항목 바꾸기
:n,ms/old/new/g          # n줄부터 m줄까지 모든 항목 바꾸기
:%s/old/new/g            # 전체 파일의 모든 항목 바꾸기
:%s/old/new/gc           # 확인을 요청하며 모든 항목 바꾸기
:1,$s/old/new/g          # :%s/old/new/g와 동일
```

## 8. Vi 환경 설정

### 8.1 Set 명령어

| 명령어 | 동작 |
|--------|------|
| `:set nu` 또는 `:set number` | 줄 번호 표시 |
| `:set nonu` | 줄 번호 숨기기 |
| `:set ai` | 자동 들여쓰기 |
| `:set noai` | 자동 들여쓰기 비활성화 |
| `:set ic` | 검색 시 대소문자 무시 |
| `:set noic` | 검색 시 대소문자 구분 |
| `:set ts=4` | 탭 간격을 4칸으로 설정 |
| `:set showmode` | 현재 모드 표시 |
| `:set all` | 모든 설정 표시 |
| `:set syntax=on` | 구문 강조 활성화 |

### 8.2 영구 설정 (~/.vimrc)

```vim
" ~/.vimrc - Vim 설정 파일
set number              " 줄 번호 표시
set autoindent          " 새 줄 자동 들여쓰기
set tabstop=4           " 탭 너비 = 4칸
set shiftwidth=4        " 들여쓰기 너비 = 4칸
set expandtab           " 탭을 공백으로 변환
set hlsearch            " 검색 결과 강조
set ignorecase          " 대소문자 무시 검색
set smartcase           " 대문자 사용 시 대소문자 구분
set showmatch           " 대응하는 괄호 강조
syntax on               " 구문 강조 활성화
set encoding=utf-8      " UTF-8 인코딩
```

### 8.3 기타 유용한 명령어

```vim
:!command                # vi를 종료하지 않고 셸 명령어 실행
:r filename              # 파일 내용을 읽어서 삽입
:r !command              # 명령어 출력 삽입
:n                       # 다음 파일로 이동 (여러 파일 편집 시)
:prev                    # 이전 파일로 이동
:split filename          # 가로로 창 분할
:vsplit filename         # 세로로 창 분할
Ctrl+w w                 # 분할된 창 간 전환
```

## 9. 모범 사례

### 9.1 생산성 팁

- `gg=G`로 전체 파일 자동 들여쓰기 (vim에서)
- `.`으로 마지막 명령 반복
- `ciw`로 내부 단어를 효율적으로 변경
- `v`, `V`, `Ctrl+v`의 비주얼 모드로 블록 선택 (vim에서)
- 방향키 대신 `hjkl` 탐색 습득

### 9.2 Vi vs. Nano vs. Emacs

| 기능 | Vi/Vim | Nano | Emacs |
|------|--------|------|-------|
| 학습 곡선 | 가파름 | 쉬움 | 가파름 |
| 어디서나 사용 가능 | 예 | 대체로 | 대체로 |
| 확장성 | 높음 (Vim) | 낮음 | 매우 높음 |
| 모달 편집 | 예 | 아니오 | 아니오 |
| 속도 (숙련 사용자) | 매우 빠름 | 보통 | 빠름 |

### 9.3 Vim 실습 과제

강의에는 vi 실습 과제가 포함되어 있다:
- **C 프로그램** 또는 **Python 프로그램**을 작성하고 실행하여 구구단(1~9단)을 출력
- `gcc`를 사용한 C 코드 컴파일과 `python3`을 사용한 Python 코드 실행 연습

## 요약

| 개념 | 핵심 사항 |
|------|-----------|
| Vi 모드 | 명령 모드 (기본), 입력 모드 (`i/a/o`), 마지막 줄 모드 (`:`) |
| 종료 | `:q` (종료), `:wq` (저장+종료), `:q!` (강제 종료), `ZZ` (저장+종료) |

| 삭제 | `x` (문자), `dw` (단어), `dd` (줄), `D` (줄 끝까지) |
| 복사/붙여넣기 | `yy` (줄 복사), `p` (뒤에 붙여넣기), `P` (앞에 붙여넣기) |
| 검색 | `/pattern` (앞으로), `?pattern` (뒤로), `n/N` (다음/이전) |
| 바꾸기 | `:%s/old/new/g` (전체), `:%s/old/new/gc` (확인 포함) |
| 설정 | `:set nu` (줄 번호), `~/.vimrc` (영구 설정) |
